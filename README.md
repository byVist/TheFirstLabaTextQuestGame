# Текстовый квест «Черное Солнце»

## Описание
Даонный проект - простой текстовый квест в стиле игры «Сталкер». Проект предназначен для демонстрации работы репозитория
## Разработка
Проект реализован в Visual Studio 2019 на языке программирования C#. Основная механика игры основана на взаимодействии игрока с виртуальным окружением через текстовые команды.
