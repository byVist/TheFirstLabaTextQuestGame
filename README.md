# Текстовый квест «Черное Солнце»

## Описание
Данный проект - простой текстовый квест в стиле игры «Сталкер». Проект предназначен для демонстрации работы репозитория
## Разработка
Проект реализован в Visual Studio 2017 на языке программирования C#. Основная механика игры основана на взаимодействии игрока с виртуальным окружением через текстовые команды.

## Авторы
Студент ИСП(11)-24-1 Замаоратский Данил Максимович.

## Текущие возможности
- Простое меню выбора действий.
- Возможность перемещения по зонам.
- Взаимодействие с предметами.

## Улучшение
Во второй версии игры реализуется графическое оформление истории (сопровождение картинками).

### Версия
v1.0 - Первая версия (нет картинок).
v2.0 - Вторая версия (картинки).
